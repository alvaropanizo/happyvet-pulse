# HappyVet Pulse

Lean Human-in-the-Loop IDP foundation for veterinary medical records.

## Run with Docker

```bash
docker compose up --build
```

### Endpoints

- Backend: `http://localhost:8000`
- Health check: `http://localhost:8000/health`
- Frontend: `http://localhost:5173`

### Stop and cleanup

```bash
docker compose down
```
